## What You'll See

### Empty State
- Message: "No exam schedules yet"
- This means no exams have been scheduled yet
- Schedule exams from the teacher dashboard first

### With Scheduled Exams
- Expandable list of dates
- Each date shows all exams scheduled for that day
- Each exam shows:
  - Subject code and name
  - Department
  - Exam type (IA1, IA2, MODEL)
  - Who assigned it
  - **Edit and Delete buttons**

## Common Tasks
